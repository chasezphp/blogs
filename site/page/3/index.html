    • JavaScript

    基础点：

    Control.js

    控件基类模块，该类不可以直接使用，经过继承之后，形成更加具体的按钮之类的控件才使用，可以认为就是一个控件抽象基类。

    包含如下一些自有属性：

    • type ：控件的类型，比如 Button 、 Input 、 Form 、 Calendar 等等
    • skin ：控件的皮肤，仅在初始化时设置有效，运行时不得变更
    • styleType ：控件的样式类型，用于生成各class使用，如无此属性，则使用 Control#type 属性代替
    • id ： 控件的 id
